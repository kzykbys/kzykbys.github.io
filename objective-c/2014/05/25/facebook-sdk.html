前回 で SDK を使用して、実際にコーディングする準備ができました。今回は、実際にログイン機能の作成をしてみます。Facebook ログインを実装するには、Facebook が用意してくれたログインボタンを使用する方法と、API を呼びカスタム UI を使用する方法があるのですが、今回はその前者を使います。
