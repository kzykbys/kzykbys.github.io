Appleの定期購読の情報を取ってくるRubyプログラム

2014-10-15 21:30:00 +0900
  • Tweet

iOSアプリにおいて、In-App Purchaseの定期購読のサービスを提供している際に、その購読者の情報を取得できるRubyのスクリプトをざくっと作成しました。

 1 require 'rest_client'
 2 
 3 apple_url = "https://buy.itunes.apple.com/verifyReceipt"
 4 
 5 num = 0
 6 File.open('ios_subscription_member.csv').each do |code|
 7   apple_url = "https://buy.itunes.apple.com/verifyReceipt"
 8   num = num + 1
 9   receipt.gsub!(/\n/, "")
10 
11   res = RestClient.post apple_url, {:'receipt-data' => code, :'password' => "*****"}.to_json
12   res = JSON.parse(res)
13 
14   isSand = "No"
15   
16   if res["status"] == 21007
17     # p "It should be a receipt code for SandBox"
18     isSand = "Yes"
19     apple_url = "https://sandbox.itunes.apple.com/verifyReceipt"
20     res = RestClient.post apple_url, {:'receipt-data' => code, :'password' => "*****"}.to_json
21     res = JSON.parse(res)
22   end
23 
24   stillPurchage = "No"
25   originalDate = "No info"
26   lastPurchasedDate = "No info"
27   expiresDate = "No info"
28 
29   # 購入者は現在も購読中
30   if res["latest_receipt"] != nil
31     stillPurchage = "Yes"
32     originalDate = res["latest_receipt_info"]["original_purchase_date"]
33     lastPurchasedDate = res["latest_receipt_info"]["purchase_date"]
34     expiresDate = res["latest_receipt_info"]["expires_date_formatted"]
35   end
36 
37   # 購入者は過去に購入したが、現在は解約
38   if res["latest_expired_receipt_info"] != nil && res["latest_expired_receipt_info"]["original_purchase_date"] != nil    
39     originalDate = res["latest_expired_receipt_info"]["original_purchase_date"]
40   end
41   if res["latest_expired_receipt_info"] != nil && res["latest_expired_receipt_info"]["purchase_date"] != nil
42     lastPurchasedDate = res["latest_expired_receipt_info"]["purchase_date"]
43   end
44   if res["latest_expired_receipt_info"] != nil && res["latest_expired_receipt_info"]["expires_date_formatted"] != nil   
45     expiresDate = res["receipt"]["expires_date_formatted"]
46   end  
47   
48   # Let's add receipt
49   lastline = "#{num}\t#{originalDate}\t#{lastPurchasedDate}\t#{expiresDate}\t#{stillPurchage}\t#{isSand}"
50   puts lastline 
51 end

ただ、ちょっと取得している情報が違うかもしれません。何かあったらご指摘お願いします。

csvファイルにて、レシートのハッシュキーがあるという前提です。あとは、ruby apple-script.rb >> results.tsvのように、tsvでアウトプットすれば、エクセル等にそのままコピペできます。上記のコードの中で指定されているパスワードはiTunes Connectの中で取得できます。
