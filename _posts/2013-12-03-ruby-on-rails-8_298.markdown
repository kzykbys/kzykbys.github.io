### ログアウト機能

ログアウトはすごく簡単です。コードを見れば理解できると思います。

{% highlight ruby linenos %}
#session_helper.rb
def destroy
  sign_out
  redirect_to root_url
end
{% endhighlight %}

{% highlight ruby linenos %}
# sessions_helper.rb 
def sign_out
  self.current_student = nil
  cookies.delete(:remember_token)
end 
{% endhighlight %}

これが、セッションを管理する際に作ったリスト。
